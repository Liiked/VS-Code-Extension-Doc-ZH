# 活动栏

活动栏是 VS Code 中的核心导航界面。插件可以把 [视图容器](../ux-guidelines/views.md#视图容器) 配置到活动栏。用户也可以把活动栏中的每一项拖动到其他区域如面板来自定义布局。

✔**️推荐**
- 使用和默认活动栏项目图标相匹配的图标样式
- 使用清晰、易懂的 [视图容器](../ux-guidelines/views.md#视图容器) 名称和子项。

❌**不推荐**
- 与现有图标撞衫
- 用活动栏子项打开 Webview 面板
